##  技術概要

- フロントエンド
  - Vue.js 2.6.12
- バックエンド
  - PHP 7.4.21
  - Laravel 6.20.26
  - PHPUnit(テスト)
  - nginx(Webサーバー)
  - RDS Postgres(データベース)
- インフラ
  - AWS(Route53, VPC, RDS, EC2)
  - AWS(CodePipline, S3, CodeCommit, CodeDeploy, CodeBuild)
  - Docker/docker-compose
  - CircleCI(バックエンドのCI/CD)
  - MailGrid(メール送信)
